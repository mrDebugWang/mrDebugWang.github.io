2. 博客网站本地搭建

（1）在本地任意位置新建一个文件夹，作为后续个人博客网站项目的本地存放位置；如：D:\MyBlogs\github-blogs\

（2）打开cmd窗口，切换到目录D:\MyBlogs\github-blogs\

Tips: 你可以直接在该目录中文件管理器上将文件路径改为cmd回车就可以直接进入该目录下的cmd窗口

（3）拉取hexo项目：cmd中执行hexo init

（4）编译生成静态网站：也就是生成css、js和html这些前端网站的文件。继续在cmd中执行hexo g

3. github在线搭建环境准备

（1）首先，我们需要创建一个github账号（free版就行）

（2）然后，我们新建一个repo，命名规则（强制性）为：你的账号名.github.io，类型一定要是public（不然别人访问不了哦）

（3）在本地项目文件夹下，鼠标右键打开git bash

（4）在bash窗口中分别执行如下步骤：

  • 配置git相关属性：使后续能正常链接到github

  • 配置ssh密钥，在后续链接github时，作CS匹配验证：

    • 首先，在bash窗口执行 ssh-keygen -t rsa，生成ssh公钥
    • 再执行：clip < ~/.ssh/id_rsa.pub将生成的公钥复制到粘贴板

（5）github新增ssh公钥： Settings → SSH and GPG → New SSH keys 将公钥添加上去
